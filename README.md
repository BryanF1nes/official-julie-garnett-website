# Julie Ann Garnett | Christian Author

This is the official website for Julie Ann Garnett. A Christian author who is dedicating her life to bringing biblical lessons and stories to both children and their parents around the world.
Her initial release of "Daisy's Delightful Day" is a beautiful take on the donkey that Jesus rode into Jerusalem.

## Tech Stack Used
JavaScript, HTML, CSS
Webpack for Bundling
Prettier and eslint for code consistency
